# 変化のかたち ― 未来をつくる100の型

ソーシャルイノベーションの新しい読み方。NPO法人ミラツク代表理事・西村勇也による全100回連載。

社会を少しでもよくする小さな試みの奥にある「かたち」を、人類史と知の系譜のなかで読み解いていきます。
